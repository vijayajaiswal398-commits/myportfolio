# 🌟 Interactive Personal Portfolio

A responsive, modern personal portfolio website built to showcase my technical journey as an AI-ML student at IGDTUW. This project features a clean UI, custom theme toggling, and a focus on responsive web design.

## 🚀 Live Demo
Check out the live site here: http://myportfolio-iota-five-28.vercel.app

## ✨ Key Features
- **Dynamic Theme Toggle:** Custom-built Light/Dark mode functionality using JavaScript and LocalStorage to save user preference.
- **Fully Responsive:** Optimized for mobile, tablet, and desktop views using CSS Media Queries and flexible layouts.
- **Modern UI:** Styled with a unique aesthetic, featuring custom hover effects and smooth transitions.
- **GitHub Integration:** Deployed via Vercel with automated CI/CD for continuous updates.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Styling:** Custom CSS (with some Tailwind CSS experimentation)
- **Tools:** Git, GitHub, VS Code, Vercel
